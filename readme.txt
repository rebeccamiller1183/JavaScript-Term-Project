Game Descriptions: 
Tank decoration: User can decorate a fish tank with various fish and decorations using drag and drop jquery
Coin toss: A game where the user chooses between heads and tails, flips the coin, and repeats. There will be a score counter keeping track of wins and losses
Clam roulette: User chooses between three clams, one of them causes a loss, the other two contain pearls. Scoreboard will include number of losses, current win streak, and longest win streak
